# Bible Puzzle Quest - Demo QR Code

## Option 1: Expo Development QR Code
1. Start your app with tunnel: `npx expo start --tunnel`
2. Scan the QR code that appears in the terminal

## Option 2: Local Network QR Code
If you're on the same WiFi network:
1. Start your app: `npx expo start --lan`
2. Connect using: `exp://172.20.10.3:8081`

## Option 3: Web Demo
1. Start web version: `npx expo start --web`
2. Access via browser at your local IP

## Instructions for Testing:
1. Install Expo Go app on your phone
2. Scan any QR code above
3. Test both sliding and jigsaw puzzles
4. Try the "Choose Your Challenge" modal
